# Instructions for Adding Fonts and Arc Raiders Theme

## 1. Install the Fonts

To install the required Arc Raiders font:

- **Download:** Get the Oxanium font from the following link:  
  https://www.dafont.com/oxanium.font

- **Font file:** From the downloaded font package, locate the file:  
  `Oxanium-Bold.ttf`

- **Destination folder:** Place the file in the following directory:  
  `Programs\Turing System Monitor\res\fonts\arc-raiders`

- **Check config:** Confirm the filename and path match any references in your theme configuration.

---

## 2. Add the Arc Raiders Theme

To install the Arc Raiders theme:

- **Locate theme:** Find the `Arc Raiders` folder in the GitHub repository.  
- **Copy folder:** Copy the entire folder, including all subfolders and assets.  
- **Destination folder:** Place it in the following directory:  
  `Programs\Turing System Monitor\res\themes\`
- **Folder structure:** Ensure the folder structure remains intact so the theme loads correctly.

---

## 3. Final Verification

Before launching, confirm:

- **Font present:** `Oxanium-Bold.ttf` is present in:  
  `Programs\Turing System Monitor\res\fonts\arc-raiders`

- **Theme present:** The `Arc Raiders` folder is present in:  
  `Programs\Turing System Monitor\res\themes\`

- **Config paths:** Any config files reference the correct paths.
